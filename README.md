# Cognotiv

Basic programming with Laravel 8, Vue 3 and JSON API for Blogs

## Installation

### From source

```bash
$ git clone git@github.com:teguharifudin/cognotiv.git
$ cd cognotiv/
$ chmod +x tool.sh
$ ./tool.sh build
```
